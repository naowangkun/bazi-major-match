# 八字-专业匹配规则（完整版）

## 第一步：排盘与基础分析

参见 [八字排盘规则](bazi-calendar.md) 完成四柱排盘。

确认以下基础信息：
1. 四柱八字（年柱、月柱、日柱、时柱）
2. 日主五行与阴阳
3. 各柱藏干
4. 五行统计（含藏干）

---

## 第二步：确定日主强弱与喜用神

### 日主强弱判断

| 条件 | 判定 |
|------|------|
| 月令生日主（同五行/生我）+ 全局多生扶 | 身强 |
| 月令生日主但全局克泄多 | 偏强 |
| 月令克泄日主但全局多生扶 | 偏弱 |
| 月令克泄日主 + 全局少生扶 | 身弱 |
| 全局几乎全是一个五行方向 | 从格（从强/从弱） |

### 喜用神选择原则

| 日主状态 | 喜用神方向 | 忌神方向 |
|---------|-----------|---------|
| 身强 | 克（官杀）、泄（食伤）、耗（财） | 生（印）、扶（比劫） |
| 身弱 | 生（印）、扶（比劫） | 克（官杀）、泄（食伤）、耗（财） |
| 中和 | 通关调和 | 破坏平衡的五行 |

### 调候优先原则

夏月（巳午未）出生的调候需求优先于强弱：
- 即使身强，夏月出生也多需水调候
- 即使身弱，冬月出生也多需火暖局

## 第三步：本科 vs 专科决策

**这是志愿填报的关键一步。不是分数低就走专科——有些八字天然更适合技能型发展。**

### 八字特征与学历路径匹配

| 八字特征 | 倾向 | 建议路径 |
|---------|------|---------|
| 正印旺且为喜用 | 学术型 | 优先本科，适合理论研究、考研深造 |
| 正官有力、官印相生 | 学术型 | 优先本科，适合考公考研、体制内方向 |
| 偏印旺且为喜用 | 学术型 | 本科，适合偏冷门的学术研究 |
| 食伤旺、食伤生财 | 技能型 | 本专科皆可，关键在于技术能力 |
| 七杀有制（食神制杀） | 技能型 | 应用型本科或优质专科，动手能力强 |
| 财星旺而破印 | 应用型 | 专科快车道亦可，先就业后深造 |
| 比劫重重、身强 | 实践型 | 专科实训环境有利，动手即优势 |
| 伤官透干、伤官配印 | 学术+技能 | 本科优先，兼顾技术与理论研究 |
| 从格（从强/从财/从杀/从儿） | 特殊 | 顺从八字专长方向，不拘学历 |

### 决策口诀

> **印旺走本科，食伤走技术，财旺看应用，官旺体制路。**
