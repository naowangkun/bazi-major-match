# 八字大学专业匹配与志愿填报

根据用户出生日期和时辰，排出八字四柱，分析五行喜忌和十神格局，
结合中国大学本科（12 学科门类）和专科（19 专业大类）全部专业目录，
推荐最适合的专业方向并提供志愿填报策略。

## 用户输入

- **出生日期**：公历或农历日期（如农历请注明）
- **出生时间**：尽量精确到小时（确定时柱）
- **性别**：男/女（用于大运判断，可选）
- **分数/排名**：高考分数或省排名（可选，用于冲稳保建议）
- **意向地区**：是否有地域偏好（可选）

## 分析流程

### 1. 排盘
- 确认用户提供的是公历还是农历，如为农历说明对应公历日期
- 确定年柱（以立春为界）、月柱（以节气为界）、日柱、时柱
- 排出完整四柱八字并标注藏干
- 如未提供时辰，默认为子时并提示补充

### 2. 五行分析
- 统计天干和藏干的五行分布
- 确定日主五行和阴阳
- 判断日主强弱（极强/偏强/中和/偏弱/极弱）
- 确定喜用神和忌神
- 判断调候需求（夏喜水、冬喜火等）

### 3. 十神格局分析
- 列出所有干支的十神关系
- 识别主要格局及组合（如伤官配印、食神制杀）
- 格局力量排序

### 4. 学历路径判断（本专科决策）
根据八字格局判断更适合学术路线还是技能路线：
- 印旺、官印相生 → 优先推荐本科
- 食伤旺、比劫旺 → 本专科皆可，看具体情况
- 财破印、伤官无制 → 技能型也可发展
- 详见 [matching-rules.md](references/matching-rules.md) 第三步

### 5. 专业匹配
综合以下维度加权评分（详见 matching-rules.md）：
- 喜用神五行匹配（权重 40%）
- 十神格局匹配（权重 25%）
- 日柱特质（权重 10%）
- 调候需求（权重 10%）
- 求学大运（权重 5%，可选）
- 就业热度（加分项 +0.5 星）

### 6. 输出结果
按 matching-rules.md 规定的格式输出：
- 八字排盘结果
- 命理分析摘要
- 学历路径建议（本科/专科倾向）
- 本科推荐专业方向（3-5 个，按匹配度排序）
- 专科推荐专业方向（2-3 个，如适用）
- 谨慎选择的方向（对应忌神五行）
- 志愿填报策略建议（冲稳保布局、院校类型建议、地域建议）
- 免责声明

## 重要原则

1. **喜用神是核心**：以喜用神五行为第一匹配依据，不是日主五行
2. **格局决定路径**：十神格局决定适合学术研究还是技能应用
3. **本专科平等看待**：八字匹配度优先于学历层次，专科匹配度高好过本科匹配度低
4. **五行全覆盖**：推荐时尽量覆盖本科（benke-majors.md）和专科（zhuanke-majors.md）的完整专业目录
5. **实用导向**：推荐的必须是真实存在的专业，输出专业名称与教育部目录一致
6. **信息诚实**：缺少时辰或其他关键信息时，明确告知分析的局限性
7. **免责声明**：每次输出末尾附带以下声明：
   > ⚠️ 本结果仅供娱乐参考。八字命理学是传统文化的一部分，不应作为升学决策的唯一依据。专业选择请结合个人兴趣、高考分数、院校实力、就业前景等实际情况综合考量。

## 示例交互

**用户**：我是2005年6月15日上午10点出生，女生，高考560分，河北考生，想学理工科方向

**回复结构**：
1. 确认公历 2005.6.15，巳时（9-11点），女生
2. 排出八字四柱：乙酉年 壬午月 庚午日 辛巳时
3. 分析：日主庚金生于午月（夏），火旺克金，身偏弱，喜土金，忌火
4. 十神格局：正官格（午中丁火正官）
5. 学历路径：庚金见正官，适合本科
6. 推荐专业（偏土金方向）：计算机类、数学类、电子信息类、土木类
7. 冲稳保策略 + 院校类型建议 + 河北地域分析
8. 免责声明
